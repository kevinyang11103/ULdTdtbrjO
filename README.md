# 旅游景点售票系统简介 python1034

## 项目概述

旅游景点售票系统是一个结合了前后端技术的在线票务销售平台。系统采用Python + Django + Vue.js + MySQL技术栈，实现了一套完善、易于扩展的票务管理系统。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户注册与登录
- 景点展示与搜索
- 票务购买与支付
- 订单管理
- 用户权限管理

## 系统角色

- 管理员：负责系统管理、景点信息维护、订单处理等操作。
- 用户：浏览景点信息、购买门票、管理个人信息等。

## 运行环境

- 后端：Python 3.x，Django 3.x
- 前端：Vue.js 2.x，Node.js（版本高于14）
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

1. 安装后端依赖库：Python、Django、MySQL等。
2. 安装前端依赖库：Vue.js、Node.js、cnpm等。
3. 部署前后端代码到服务器。
4. 配置数据库连接。
5. 启动后端服务。
6. 启动前端服务。
